Добро пожаловать в программу QuizRunner 1.0

Программа была созданна тремя студентами гр.ИП-715:
 Допчук Вячеслав, Мясненко Максим, Киселев Владимир.
В данной программе пользователю предоставленна возможность:
*Cоздать свой тест.
*Пройти имеющийся.

Меню программы состоит из:
=====МЕНЮ=====
1.Создать тест  W-перемещение вверх
2.Пройти тест   S-перемещение вниз
3.Помощь        E-взаимодействие
4.Выход
==============
1. В данном пункте меню, пользователь может создать свой файл с тестом, 
который будет хранится в директории CourseWork/txt, а так же из этого пунка меню, 
в файл лежащий на пути CourseWork/Tests/Tests.txt будет сделана пометка 
о новой доступной для прохождения теме.

2. При прохождении в этот пункт меню, пользователь попадет в окно выбора существующих тем,чтобы пополнить 
перечень тем смотри (1.) после выбора темы, путем ввода с клавиатуры значений не выходящих за допустимые значения. 
Пользователь будет направлен в окно прохождения тестирования.Перед прохождением теста, с пользователя будет спрошено 
его имя и фамилия, что бы хранить результат пройденный им теста. За каждый верный ответ, программа будет начислять по одному очку, 
после прохождения тестирования вы сможете посмотреть свой результат в отношении с общим числом вопросов, 
так же будет создан файл с результатом по этой теме. Результаты будут лежать в папке CourseWork/Results/*.txt 
в файлах будет хранится Имя и результат в процентах.

3. Вам нужна помощь о пунке меню "Помощь"? Рекурсивненько.

4. Exit, Выхад, Вихiд, Излаз, Utgang, Salida...

Тонкости программы: При создании теста, в момент указания правильных номеров ответов, необходимо записывать только численные значения.
При вводе правильных ответов при прохождении теста, должны указываться только численные значения, а не символьные! Иначе вы будете
отправленны в меню.%
